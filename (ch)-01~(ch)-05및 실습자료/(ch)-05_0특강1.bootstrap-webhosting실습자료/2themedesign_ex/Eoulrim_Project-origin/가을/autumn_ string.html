# 가을 스트링 컬러란 ?
당신과 조화를 이루는 색은 회색 기와 푸른 기가 살짝 도는 고명도의 은은한 파스텔 컬러입니다.
‘가을 스트링 톤’의 컬러는 흰색이나 연하고 밝은 회색이 섞여 싱그러운 느낌을 줍니다.

가을 스트링인 당신은

가을 스트링 톤은 원색이 잘 어울리는 파스텔 컬러입니다.
라이트와 뮤트 톤 사이의 무난한 중간색으로 너무 연하거나 진하지 않으면
모든 쿨톤 색이 잘 어울립니다.




당신의 팔레트를 채워드려요

선명하고 화려한 색상보단 은은하면서도 맑고 밝은 쿨 컬러가 베스트로 꼽힌다.
이러한 특징 때문에 여름 쿨톤의 경우 짙고 딥한 메이크업보단 맑고 가벼운 메이크업이 잘 어울린다.




베스트 스타일링

메이크업 - 아이
옅게 진하고 쨍한 컬러의 여름 쿨톤 컬러를 전체적으로 바르면 어울려요

- 치크
여리여리하면서도 생기를 줄 수 있는 핑크 블러셔가 잘 어울리고
특히 질감이 부드럽고 텁텁하지 않고 맑게 표현되는 것이 가을 스트링와 잘 어울려요

- 립
또렷한 핫핑크 립이 완전 찰떡인 가을 스트링 컬링고, 립을 강조한 포인트 메이크업이 잘 어울려
화사하면서도 발랄한 핑크와 글로스로 광택감을 주면 컬러 그대로 올라와서 더 이쁘게 보여요





가을 스트링 위한 코디 추천

주로 코디할 때 쿨톤 컬러를 사용하는 것을 제외하고 제 1순위로 신경써야 하는 것은
바로 대비감입니다. 대비감을 통해 생기있게 연출할 수 있습니다.
