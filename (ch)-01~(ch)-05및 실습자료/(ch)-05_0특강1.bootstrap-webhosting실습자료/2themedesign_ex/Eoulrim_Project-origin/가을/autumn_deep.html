# 가을 딥 컬러란 ?
당신과 조화를 이루는 색은 회색 기와 푸른 기가 살짝 도는 고명도의 은은한 파스텔 컬러입니다.
‘가을 딥 톤’의 컬러는 흰색이나 연하고 밝은 회색이 섞여 싱그러운 느낌을 줍니다.

가을 딥인 당신은

피부색이 투명하고 붉은 기를 가지고 있어서 인상은 차분하면서도 청량감이 넘치는 사람입니다.
고명도 • 고체도의 블루베이스의 색상이 어울립니다.
